# AMSGeekCodingChallenge
A simple deck of cards program to test coding proficiency!

Below is mock-documentation for how to use the program and methods created.</br>

<h3>createDeck()</h3>
<h5>Description</h5>
<p>Creates a standard 52 card deck in face value order</p></br>

<h3>shuffle(deck, timesToShuffle)</h3>
<h5>Description</h5>
<p>Shuffles a given deck of cards (? times)</p>
<h5>Parameters</h5>
<p>deck - A Deck object containing a list of card objects</p>
<p>(Optional) timesToShuffle - an integer representing the number of times to shuffle a deck</p></br>

<h3>dealOneCard(deck, cardsToDeal)</h3>
<h5>Description</h5>
<p>Deals a card from a given deck of cards (? times)</p>
<h5>Parameters</h5>
<p>deck - A Deck object containing a list of card objects</p>
<p>(Optional) cardsToDeal - an integer representing the number of cards to deal from the deck</p>
